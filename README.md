# Nugget AI 1.1.4                                                                                                                                    
                                                                                                                                                            
  The headline of this release: **your interview data now lives wherever you work.** We've also leveled up team management and connection options.          
   
  ## ✨ What's new                                                                                                                                          
                  
  ### 🔌 MCP Server — Nugget, anywhere you use AI                                                                                                           
                  
  Nugget now has a remote **Model Context Protocol server**, which means Claude, Cursor, and any other MCP-compatible AI tool can read your interviews,     
  nuggets, themes, priorities, specs, and reports directly.
                                                                                                                                                            
  Ask Claude *"what are the top 5 pain points from last month's interviews?"* and get a real answer pulled from your actual data — no copy-paste, no        
  exports.
                                                                                                                                                            
  - Generate scoped access tokens from **Settings → Connections**
  - Granular permissions per resource (read interviews, write specs, etc.)
  - Hosted infrastructure with audit logging and rate limiting                                                                                              
  - Tools cover: interviews, nuggets, participants, themes, priorities, specs, scripts, tasks, reports
                                                                                                                                                            
  ### 👥 Team & Member Management
                                                                                                                                                            
  New member management built into Settings. Invite teammates, manage roles, and keep your interview library shared across your product team.               
   
  ### 🔗 Connections Hub                                                                                                                                    
                  
  A new **Connections** page in Settings where you can:

  - Manage your MCP tokens
  - Express interest in upcoming integrations (Linear, Jira, Notion, Slack, and more) — your votes shape what we build next
                                                                                                                                                            
  ### 🪄  Smarter Insight Reports
                                                                                                                                                            
  - Improved insight report generation with better evidence linking
  - Refined theme generation with cleaner clustering and edge cases handled
                                                                                                                                                            
  ### 🔧 Under the hood
                                                                                                                                                            
  - New deeplink handling for smoother navigation from external tools
  - Tiptap ↔ Markdown conversion shared across the app and MCP server (specs and reports look the same wherever they're rendered)
  - Embedding query pipeline improvements                                                                                                                   
   
  ## 💛 Coming soon                                                                                                                                         
                  
  We're already heads-down on native integrations for Linear, Jira, and Slack — the Connections page is how you tell us which ones to ship first.
